## 🚀 Quick Start

### Prerequisites
- Node.js 18.0.0 or higher
- npm 10.0.0 or higher

### Installation

```bash
# Clone the repository
git clone https://github.com/Zion-Holdings/zion.app.git

# Navigate to the project directory
cd zion.app

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Available Scripts

#### 🎯 Enhancement Scripts
```bash
# Run all enhancements
npm run enhance:all

# Performance optimizations
npm run enhance:performance

# SEO improvements
npm run enhance:seo

# UX enhancements
npm run enhance:ux

# Security improvements
npm run enhance:security
```

#### 🔧 Development Scripts
```bash
# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start

# Run tests
npm test

# Run linting
npm run lint

# Type checking
npm run type-check
```

#### 📊 Analysis Scripts

npm run audit:security

# Accessibility testing
npm run test:accessibility

# Performance testing
npm run test:performance
```

## 🏗️ Architecture

### Technology Stack
- **Framework**: Next.js 15.5.3
- **Language**: TypeScript 5.9.2
- **Styling**: Tailwind CSS 3.4.17
- **Animation**: Framer Motion 11.0.3
- **Icons**: Lucide React 0.544.0
- **State Management**: Redux Toolkit 2.9.0
- **Testing**: Jest 30.1.3
- **Performance**: Web Vitals 5.1.0

### Project Structure
```
zion.app/
├── app/                    # Next.js App Router
│   ├── page.tsx           # Main page (enhanced)
│   ├── enhanced-page.tsx  # Enhanced version with lazy loading

├── components/            # React components

├── automation/            # Automation systems
├── content/               # Content management

## 🔧 Configuration

### Enhancement Configuration
The application uses `enhancement.config.json` to configure various enhancement features:

```json
{
  "performance": {
    "lazyLoading": true,
    "codeSplitting": true,
    "bundleAnalysis": true,
    "imageOptimization": true
  },
  "seo": {
    "metaTags": true,
    "structuredData": true,
    "sitemap": true,
    "openGraph": true
  },
  "ux": {
    "loadingStates": true,
    "errorBoundaries": true,
    "accessibility": true,
    "darkMode": true
  }
}
```

### Environment Variables
```bash
# Performance monitoring
NEXT_PUBLIC_ANALYTICS_ID=your_analytics_id

# SEO configuration
NEXT_PUBLIC_SITE_URL=https://zion.app
NEXT_PUBLIC_SITE_NAME="Zion Tech Group"

# Performance optimization
NEXT_PUBLIC_ENABLE_PERFORMANCE_MONITORING=true
```
