## 🛠️ Development Setup

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Git

### Installation
```bash
git clone https://github.com/Zion-Holdings/zion.app.git
cd zion.app
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
npm start
```
