# Zion Tech Group Website

A modern, high-performance website built with React, TypeScript, and Vite, featuring AI-powered solutions and technology consulting services.

## 🚀 Features

- **Modern Tech Stack**: React 18, TypeScript, Vite, Tailwind CSS
- **Performance Optimized**: Code splitting, lazy loading, bundle optimization
- **Mobile Responsive**: Fully responsive design with mobile-first approach
- **SEO Optimized**: Meta tags, structured data, sitemap support
- **Accessibility**: ARIA labels, keyboard navigation, screen reader support
- **PWA Ready**: Web app manifest, service worker support
- **Error Handling**: Comprehensive error boundaries and monitoring

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, Framer Motion
- **Icons**: Lucide React
- **SEO**: React Helmet Async
- **Routing**: React Router DOM
- **Build Tool**: Vite with Terser minification

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run start
```

## 🏗️ Project Structure

```
src/
├── components/          # Reusable UI components
│   ├── layout/         # Layout components (Header, Footer, etc.)

├── pages/              # Page components
├── utils/              # Utility functions and helpers
├── styles/             # Global styles and CSS
```
