## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- Yarn or npm
- Git

### Installation
```bash
# Clone the repository
git clone https://github.com/ziontechgroup/zion-app.git

# Navigate to project directory
cd zion-app

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Environment Variables
Create a `.env` file with the following variables:
```env
VITE_API_URL=your_api_url
VITE_OPENAI_API_KEY=your_openai_key
VITE_GOOGLE_ANALYTICS_ID=your_ga_id
```

## 🔧 Customization

### Adding New Services
1. Create service page in `src/pages/services/`
2. Add route in `src/App.tsx`
3. Update navigation in `src/layout/AppHeader.tsx`
4. Add to services overview in `src/pages/ServicesOverview.tsx`
5. Update footer links in `src/components/EnhancedFuturisticFooter.tsx`

- Uses Tailwind CSS with custom color schemes
