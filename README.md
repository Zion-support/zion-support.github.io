## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ziontechgroup/zion-website.git
   cd zion-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

4. **Build for production**
   ```bash
   npm run build
   # or
   pnpm build
   ```
