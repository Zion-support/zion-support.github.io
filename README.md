## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn package manager

### Installation
```bash
# Clone the repository
git clone https://github.com/ziontechgroup/zion-app.git

# Navigate to project directory
cd zion-app

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
