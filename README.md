# Zion Tech Group - Enterprise AI & IT Solutions

A modern, responsive web application showcasing Zion Tech Group's comprehensive suite of AI solutions, micro SaaS development, and enterprise IT services.

## 🚀 Features

### Core Functionality
- **Modern UI/UX**: Responsive design with smooth animations and transitions
- **Service Showcase**: Comprehensive display of 20+ AI and IT services
- **Contact Integration**: Multiple contact methods and forms
- **SEO Optimized**: Complete metadata, sitemap, and robots.txt
- **Performance Optimized**: Fast loading with optimized assets
- **Error Handling**: Comprehensive error boundaries and 404 pages
- **Accessibility**: ARIA labels, keyboard navigation, and screen reader support

### Technical Stack
- **Framework**: Next.js 15.5.2 with App Router
- **Styling**: Tailwind CSS with custom animations
- **Icons**: Heroicons for consistent iconography
- **TypeScript**: Full type safety throughout
- **Linting**: ESLint with comprehensive rules
- **Build**: Optimized production builds

## 📁 Project Structure

```
app/
├── components/          # Reusable UI components
│   ├── Header.tsx      # Navigation header
│   ├── Footer.tsx      # Site footer
│   └── ...
├── services/           # Service pages
│   ├── ai-services/    # AI service pages
│   ├── micro-saas/     # Micro SaaS pages
│   └── ...
├── globals.css         # Global styles and animations
├── layout.tsx          # Root layout with metadata
├── page.tsx            # Homepage
├── loading.tsx         # Loading page
├── error.tsx           # Error boundary
├── not-found.tsx       # 404 page
├── sitemap.ts          # Dynamic sitemap
├── robots.ts           # Robots.txt
├── viewport.ts         # Viewport configuration
└── analytics.ts        # Analytics utilities
```

## 🛠️ Development

### Prerequisites
- Node.js 18.0.0 or higher
- npm 8.0.0 or higher

### Installation
```bash
npm install
```

### Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Start Production Server
```bash
npm start
```

### Linting
```bash
npm run lint
```

### Type Checking
```bash
npm run type-check
```
