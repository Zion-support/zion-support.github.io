## 🚀 Quick Start

### 1. Start the Ultimate System

```bash
# Start the complete intelligent PM2 system
npm run pm2:ultimate

# Or start manually
pm2 start ecosystem.ultimate-intelligent-v4.cjs
```

### 2. Monitor the System

```bash
# View system status
npm run pm2:status

# View logs
npm run pm2:logs

# Monitor in real-time
npm run pm2:monit

# View health dashboard
npm run system:health
```

### 3. Run AI Analysis

```bash
# Run all AI analysis
npm run ai:all

# Individual AI components
npm run ai:quality      # Code quality analysis
npm run ai:security     # Security intelligence
npm run ai:performance  # Performance prediction
npm run ai:test         # Intelligent testing
npm run ai:deploy       # Deployment strategy
```

## 🔧 Troubleshooting

### Common Issues

1. **Process not starting**: Check logs with `pm2 logs <process-name>`
2. **High memory usage**: Run `npm run resource:optimize`
3. **Performance issues**: Run `npm run performance:analyze`
4. **Security concerns**: Run `npm run security:scan`
5. **Test failures**: Run `npm run test:ai`

### Log Locations

- **Main logs**: `./logs/`
- **PM2 logs**: `./logs/pm2/`
- **Process logs**: `./logs/<process-name>.log`
- **Error logs**: `./logs/<process-name>-error.log`

### Reset System

```bash
# Stop all processes
npm run pm2:stop

# Delete all processes
npm run pm2:delete

# Restart system
npm run pm2:ultimate
```
