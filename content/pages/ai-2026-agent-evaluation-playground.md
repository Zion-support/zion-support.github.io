# Agent Evaluation Playground

Spin up reproducible scenarios, datasets, and harnesses to iterate quickly and safely. Connect to live policies to preview production gates.

## Features

- Scenario templates with ground truth and scoring
- Local harness with deterministic seeds and tracing
- Policy-as-code preview and gate simulation
## Quick start

1. Select scenario suite and baseline
2. Run local evals with tracing enabled
3. Compare against thresholds; iterate prompts/tools
4. Export evidence to the evaluation hub
