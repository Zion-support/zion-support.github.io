## Detailed Curriculum

### Day 1: Foundations & Architecture (8 hours)

#### Session 1: AI Agent Fundamentals (2 hours)
*Morning: 9:00 AM - 11:00 AM*

- **What are AI agents?** Types, capabilities, and limitations
- **Agent architectures**: Reactive, goal-based, learning agents
- **Production readiness**: The 7 critical characteristics
- **Use case identification**: When to use agents vs. traditional automation
- **ROI framework**: Calculating expected business value

**Hands-On Exercise**: Evaluate 3 use cases from your organization for agent suitability

#### Session 2: Technology Stack Deep Dive (2 hours)
*Late Morning: 11:15 AM - 1:15 PM*

- **Foundation models**: GPT-4, Claude, Llama, Gemini comparison
- **Vector databases**: Pinecone, Weaviate, Qdrant, pgvector
- **Orchestration frameworks**: LangChain, LlamaIndex, LangGraph
- **Observability tools**: LangSmith, W&B, Arize, custom solutions
- **Infrastructure**: Compute requirements, cost analysis

**Workshop Activity**: Design technology stack for your use case

*Lunch Break: 1:15 PM - 2:15 PM*

#### Session 3: Architecture Patterns (2 hours)
*Afternoon: 2:15 PM - 4:15 PM*

- **The agent loop**: Perceive → Reason → Act → Learn
- **Memory architectures**: Working, episodic, semantic memory
- **Tool integration patterns**: APIs, databases, external systems
- **Multi-agent systems**: Orchestration and collaboration
- **Failure handling**: Retries, fallbacks, escalation

**Hands-On Exercise**: Design agent architecture for your use case

#### Session 4: Security & Compliance (1.5 hours)
*Late Afternoon: 4:30 PM - 6:00 PM*

- **Security threats**: Prompt injection, data leakage, adversarial attacks
- **Defense strategies**: Input validation, output sanitization, guardrails
- **Compliance requirements**: GDPR, SOC 2, HIPAA, industry-specific
- **Audit and explainability**: Logging, tracing, decision transparency
- **Cost controls**: Budget limits, rate limiting, optimization

**Workshop Activity**: Identify security and compliance requirements for your project

### Day 2: Hands-On Development (8 hours)

#### Session 5: Building Your First Agent (3 hours)
*Morning: 9:00 AM - 12:00 PM*

**Live Coding Workshop**: Build a production-ready customer service agent

- **Step 1**: Project setup and infrastructure
- **Step 2**: Implement the agent loop
- **Step 3**: Add memory and context management
- **Step 4**: Integrate tools (database, APIs, external services)
- **Step 5**: Add observability and logging

**Your Turn**: Begin building your team's agent (with instructor support)

*Lunch Break: 12:00 PM - 1:00 PM*

#### Session 6: Advanced Techniques (2.5 hours)
*Afternoon: 1:00 PM - 3:30 PM*

**Prompt Engineering Mastery**
- The CRISP framework: Context, Role, Instruction, Structure, Prompt
- Few-shot learning and in-context learning
- Chain-of-thought and self-consistency
- Prompt compression and optimization
- Evaluation and iteration

**Hands-On Exercise**: Optimize prompts for your agent

**Memory & Context Management**
- Context window management strategies
- Semantic search with vector databases
- Conversation summarization and compression
- Long-term memory patterns
- Context relevance scoring

**Hands-On Exercise**: Implement advanced memory for your agent

#### Session 7: Tool Integration & Testing (2.5 hours)
*Late Afternoon: 3:45 PM - 6:15 PM*

**Tool Integration Patterns**
- Function calling vs. tool use frameworks
- API authentication and error handling
- Rate limiting and retry logic
- Transaction management
- Human-in-the-loop patterns

**Testing Strategies**
- Unit testing agent components
- Integration testing workflows
- Adversarial testing for security
- Load testing and performance
- Regression testing from production issues

**Hands-On Exercise**: Write comprehensive tests for your agent

### Day 3: Deployment & Operations (8 hours)

#### Session 8: Deployment Strategies (2 hours)
*Morning: 9:00 AM - 11:00 AM*

- **Deployment approaches**: Shadow, assisted, autonomous modes
- **Rollout strategies**: Canary, blue-green, gradual rollout
- **Infrastructure**: Kubernetes, serverless, managed services
- **CI/CD pipelines**: Automated testing and deployment
- **Rollback procedures**: When and how to roll back

**Workshop Activity**: Create deployment plan for your agent

- Continuous cost monitoring

**Hands-On Exercise**: Set up monitoring for your agent

*Lunch Break: 1:15 PM - 2:15 PM*
