### Quick Start Guide

```python
from nas_framework import NASEngine, SearchSpace

# Define search space
search_space = SearchSpace(
    primitives=['conv3x3', 'conv5x5', 'max_pool', 'skip'],
    layers=20,
    channels=[32, 64, 128, 256]
)

# Configure NAS engine
nas_engine = NASEngine(
    search_strategy='reinforcement_learning',
    objective=['accuracy', 'latency'],
    budget=1000  # GPU hours
)

# Run architecture search
best_architecture = nas_engine.search(
    train_data=train_loader,
    val_data=val_loader,
    search_space=search_space
)

# Train final model
model = best_architecture.build()
model.train(train_data, epochs=100)
```
