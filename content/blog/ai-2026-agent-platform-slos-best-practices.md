# AI 2026: Agent Platform SLOs — Best Practices

SLOs align engineering with outcomes. These SLOs keep agent platforms safe, effective, and affordable.

## Recommended SLOs

- Quality: ≥ 95% pass on critical scenarios
- Safety: 0 critical policy violations; < 1% red flags
- Latency p95: < 800 ms for interactive paths
- Cost: budget per task and per session within targets

## Implementation Notes

- Tie SLOs to eval suites and dashboards
- Break down by capability (retrieval, routing, tool use)
- Alert on burn rates and error budgets
