# Reliable Agent Ops — Budgets, Approvals, and Live Evals

This operational playbook shows how organizations can scale agent workloads safely. Apply budget controls to tools, require approvals for high‑risk operations, and run live evaluations on every change to prevent regressions.

## The Safety Trifecta

- Budgeted execution with scoped limits
- Human‑in‑the‑loop approvals for sensitive actions
- Continuous evals with automatic rollback

## Implementation Steps

1. Define tool budgets and risk tiers
2. Enforce approvals by tier with SLAs
3. Run PR policy tests and deploy with canaries
4. Trigger automatic rollback on KPI degradation

## KPIs to Track

- Cost per successful task
- Failure rate by risk tier
- Mean time to rollback
