This starter kit provides a practical blueprint to visualize agent platform health:

- Core SLOs: availability, latency P95, incident MTTR, eval pass rate, cost per task
- Sources: traces, evaluation runs, policy decisions, incident records
- Views: exec overview, on‑call, reliability, and cost optimization

Includes: schema, sample datasets, dashboard JSON, and implementation notes for common stacks.
