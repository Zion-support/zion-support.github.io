## Technical Architecture

### Edge AI Stack

```
Store Level (5,000+ locations)
├── Edge Devices (NVIDIA Jetson AGX Orin)
│   ├── Computer Vision Models
│   ├── Recommendation Engine
│   ├── Inventory Tracking
│   └── Loss Prevention AI
├── Local Processing
│   ├── Real-time Inference (< 5ms)
│   ├── Data Preprocessing
│   ├── Local Storage (7 days)
│   └── Privacy-Preserving Analytics
└── Connectivity
    ├── 5G Backup
    ├── Edge-to-Edge Mesh
    └── Cloud Sync (summary data only)

Regional Hubs (50 locations)
├── Edge Data Centers
├── Model Training & Updates
├── Aggregated Analytics
└── Compliance Management

Central Cloud (Minimal)
├── Global Dashboard
├── Model Repository
├── Long-term Storage
└── Business Intelligence
```
