# Safe Agent Rollouts Checklist (2026)

Shipping agentic features safely requires staged rollouts, evidence-first evaluation, and clear kill‑switches.

- Preflight gates aligned to SLIs/SLOs
- Canary cohorts with policy-aware tool permissions
- Online evaluation scorecards and alert thresholds
- Automated rollback tied to incident signals

Use this checklist to move from dev to prod with confidence.
