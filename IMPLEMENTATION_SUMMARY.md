## 🛠️ Management & Usage

### Easy Management
```bash
# Start the system
./manage-automation.sh start

# Check status
./manage-automation.sh status

# View logs
./manage-automation.sh logs

# Monitor processes
./manage-automation.sh monitor

# View reports
./manage-automation.sh reports
```

### PM2 Commands
```bash
# Direct PM2 management
pm2 status                    # View all processes
pm2 logs                     # View all logs
pm2 monit                    # Monitor dashboard
pm2 restart all              # Restart all processes
pm2 stop all                 # Stop all processes
```
