# Agent Incident Postmortems — Evidence‑First Templates

A lightweight, repeatable postmortem flow for agent incidents using verifiable traces, policy events, and eval scorecards.

## Template outline

1. Incident summary and customer impact
2. Evidence bundle: traces, artifacts, and policy denials
3. Root cause via causal chain reconstruction
4. Preventive controls: gates, runbooks, approvals
5. SLIs/SLOs affected and regression tests added

Adopt these templates to cut MTTR and drive systemic fixes.
