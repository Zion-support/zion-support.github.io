Guardrails must be observable to be reliable. We describe a metrics set for runtime tool‑use safety:

- Policy check throughput and failure reasons
- Tool arbitration outcomes and fallback rates
- Latency budgets under guardrail load

Includes a Prometheus/Grafana starter dashboard and alerting recommendations for production systems.
