# Runtime Policy Controls for Safer Tool Use

Agent safety improves when policy is enforced at runtime, near tools and data.

## Principles

- Least privilege adapters and tiered permissions
- Contextual approvals for high‑risk actions
- Continuous evaluation of outputs for policy compliance

## Starter controls

- Red‑team prompts as preflight checks
- Guardrail functions with signature policies
- Evidence traces for audits and incident response

These controls reduce harm while preserving iteration speed.
